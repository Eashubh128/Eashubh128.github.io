# Task: Build Portfolio Website

## Goal
Create a modern, responsive portfolio website for a frontend developer with a dark tech theme and neon accents.

## Files to Create
- `lib/data.js` — centralized placeholder content
- `app/layout.js` — root layout
- `app/page.js` — main page composing all sections
- `app/globals.css` — Tailwind + theme
- `components/Navbar.js` — sticky responsive nav
- `components/Hero.js` — hero with typewriter + CTAs
- `components/About.js` — bio + stats
- `components/Skills.js` — tech grid
- `components/Projects.js` — project cards
- `components/Experience.js` — timeline
- `components/Contact.js` — form + socials
- `components/Footer.js` — footer + back-to-top

## Acceptance Criteria
- [ ] Website loads on preview URL
- [ ] Fully responsive (mobile, tablet, desktop)
- [ ] Dark tech theme (#0a0a0f bg + neon purple/blue/cyan accents)
- [ ] Navbar with working mobile hamburger menu
- [ ] Smooth-scroll navigation between sections
- [ ] Hero with typewriter title effect + CTA buttons
- [ ] About section with bio + animated stats
- [ ] Skills grid with tech icons
- [ ] Projects cards responsive with hover effects
- [ ] Experience vertical timeline
- [ ] Contact form (name, email, message) + social links
- [ ] Footer with quick links + back-to-top
- [ ] Scroll-triggered animations (Framer Motion)
- [ ] All placeholder content in lib/data.js
- [ ] No console errors
- [ ] Production build succeeds (npm run build)
- [ ] Background service uses `npm run start` (not `npm run dev`)
