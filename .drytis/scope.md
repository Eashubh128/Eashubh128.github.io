# Developer Portfolio — Scope

## Features (In Scope)
1. **Navbar** — sticky, responsive, hamburger menu on mobile, smooth-scroll links
2. **Hero** — name, title (typewriter), tagline, CTA buttons, animated tech background
3. **About** — bio paragraph, profile photo placeholder, stats counters
4. **Skills** — tech stack grid with icons and category labels
5. **Projects** — responsive project cards, hover effects, tech tags, links
6. **Experience** — vertical timeline of work history
7. **Contact** — contact form (name, email, message) + social links
8. **Footer** — copyright, quick links, back-to-top button

## Out of Scope
- Backend / API / database
- Contact form submission (UI only)
- Blog section
- CMS integration
- Dark/light mode toggle (dark only)
- i18n

## Phases
1. Scaffold project + install deps + config Tailwind
2. Centralized data file + globals.css + theme setup
3. Navbar + Hero
4. About + Skills
5. Projects + Experience
6. Contact + Footer
7. Wire up page, animations, responsive polish
8. Infrastructure gate + verification
